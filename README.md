# Section Header Normalization in Biomedical Articles using Transformers

Please refer to the slides for [Section Header Normalization in Biomedical Articles using Transformers](https://amia.secure-platform.com/symposium/gallery/rounds/82021/details/19558).

## Repository Structure

### Modeling & Training
* `SuperviseLearning.ipynb` / `SuperviseLearning_test.ipynb`: Notebooks for training the supervised neural network models.
* `average_embedding.ipynb`: Implementation of the Average Embedding Similarity classification method.
* `pretrained_bert.ipynb`: pre-trained encoder.

### Evaluation 
* `comparison.ipynb`: Compare results from Embedding similarity and Neural network models
